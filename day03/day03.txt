1，使用SSH方式关联远程仓库
  关联远程仓库有两种：HTTPS，SSH
  而HTTPS在每次推送时，都需要输入用户名与密码，比较麻烦
  配置SSH：
    查看本机是否有密钥
      密钥包含两个：公钥，私钥
      创建生成密钥：
        ssh-keygen -t rsa -C“邮箱地址”
      打开公钥，复制里面的内容
      点击GitHub的右上角个人头像--->设置
      ---> SSH和GPG密钥
      --->新的SSH密钥
      --->标题中填写名称（可以随便写），将复制的内容粘贴进键中
      --->添加SSH密钥

2，拉取数据
  将远程仓库的内容下载到本地
  1.先进入目标文件夹，将其初始化为本地仓库
    使用命令git pull远程仓库地址master
  2.进入文件夹，无需初始化本地仓库，使用克隆命令
    git clone远程仓库地址
    该命令下载的就是一个完整的仓库文件夹

AngularJs
1.什么是angularjs？
  是一个前端框架，AngularJs是以数据和逻辑为核心（驱动）的
  案例1：
    01_rumen.html
    在文本框中输入数据，文本框的下方显示文本框中输入的数据

2. angularjs的表达式
  {{表达式}}
  表达式可以包含变量，算数运算，逻辑运算，关系运算，三目运算，字符串拼接
  02_biaodashi.html
MVC。设计模式
  M：Model模型层，数据层，用于跟数据库进行操作
  V：查看视图层，用于展示数据，一般通过html展示
  C：控制器控制层，用于模型层与视图层之间的连接
4. AngularJs的模块化概念（模块）
  AngularJs构建一个应用程序是以模块化的方式组织的
  03_module.html
  步骤：
    1）引入angularjs文件
      <script src =“../ libs / angular.min.js”> </ script>
    2）定义纳克应用内
      定义应用名称
      在身体标签添加属性纳克应用内=“应用程序”
      <body ng-app =“app”>
    3）定义模块
      var app = angular.module（“app”，[]）;
    4）定义控制器
      app.controller（ “CTRL”，[ “$范围”，功能（$范围）{}]）
    5）编写视图（挖坑填坑）

    练习：
      1.新建html文件.
      2.定义模块，定义控制器
      3.在控制器中绑定数据
          字符串，数字，数组，对象
      4.在视图层中将这些数据全部展示出来
      5.显示数组中的第一个元素
      6.显示对象中键为名称的值
